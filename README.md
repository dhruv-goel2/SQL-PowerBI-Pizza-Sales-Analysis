# 🍕 Pizza Sales Analysis | SQL + Power BI

## 📌 Project Overview

This project analyzes pizza sales data using SQL and Power BI to identify sales trends, customer purchasing behavior, and key business insights. The interactive dashboard provides KPI tracking and visual analysis to support data-driven decision-making.

---

## 🛠 Tools & Technologies

- SQL
- Power BI
- Excel

---

## 📊 Key Performance Indicators (KPIs)

- Total Revenue
- Total Orders
- Total Pizzas Sold
- Average Order Value
- Average Pizzas Per Order

---

## 📈 Dashboard Features

- Revenue Trend Analysis
- Daily & Monthly Sales
- Sales by Pizza Category
- Sales by Pizza Size
- Top & Bottom Selling Pizzas
- Orders by Day and Month
- Interactive Filters & Slicers

---

## 💡 Business Insights

- Identified top-performing pizza categories.
- Analyzed customer ordering patterns.
- Found peak sales days and months.
- Compared pizza sizes based on revenue.
- Measured overall business performance using KPIs.
